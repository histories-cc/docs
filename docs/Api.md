# API

For API is used [GraphQL](https://graphql.org/) because it's easy to use and provides types.

Try it yourself on **[GraphQL playground](https://www.histories.cc/api/graphql)**

## Query example

**Query:** 
```
query {
  hello
}
```

**Response:**
```
{
  "data": {
    "hello": "Hello"
  }
}
```
